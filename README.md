# S3-Static-Website-Hosting
# Amazon S3 Static Website
  1. create an S3 bucket
  2. Enable website and set index.html and error.html
  3. upload Sample HTML page to index.html
  4. upload Error HTML page for 404 errors
  5. set Bucket Policy to allow public access to GetObject from the bucket
  6. Access s3 URL on browser
Cleanup script to remove s3 bucket created.
